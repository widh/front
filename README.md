World of Wldh
=============

Aims for supporting *screen reader*, *no-js environment*.

<br/>

#### Compability Information

###### Desktop

+ Safari `10+`
+ Edge `16+` `without transition effects (edgeHTML)`
+ Chrome `49+`
+ Firefox `44+` `without hover effects (44-63)`

###### Mobile

+ Android `6+`
+ iOS `10+`
